Python and Flask Explanations

Branch

A Git branch lists the user's project branches. The branch that is highlighted designates the branch that is currently in use. A user can type and develop code on a branch. This is a great way for users to make edits in a code program without damaging the finished project. It is generally advised for users not to work on the master branch (main branch). It is advised to work on a new branch and merge to the master once everything is confirmed as working.

Pytest

Pytest allows a coder to run a test on their own code to make sure their code works. In this website, the following code was used to test the website. The below code test to check if the home page is working. If the test reads Home in the home page, the test will pass.
def test_request_index(client):
"""This makes the index page"""
response = client.get("/")
assert response.status_code == 200
assert b"Home" in response.data
